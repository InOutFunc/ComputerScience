## 跳跃游戏II

## 思路

a1 -> a2~a5

假设a1能到达的范围是a2~a5，下一步应该跳哪个地方呢

看a2, a3, a4, a5下一步能跳的最远距离，哪个远，a1就跳哪里

## 遍历

```c++
for (int i = 0; i < n - 1; i++) {
    
}
```

在每个位置上做的事情：

1、更新能到达的最远距离

2、判断此次的范围是否结束，更新新的范围，增加跳的距离。若 i == last，则说明上一跳不符合条件。还需要继续

## 边界条件

初始跳的范围是`0`，从`i = 0`开始，`i = 0`结束。

此时res++了

n - 1位置说明到达终点了

## 判断下一跳

```c++
for (int i = 0; i < n - 1; i++) {
    cur = max(cur, i + nums[i]);
    if (i == last) {
        last = cur;
        res++;
        if (cur >= n - 1) {
            break;
        }
    }
}
```
